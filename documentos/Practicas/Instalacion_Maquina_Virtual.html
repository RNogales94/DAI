Instalación fácil de una Máquina Virtual para las prácticas

Para las prácticas del curso, utilizaremos una máquina virtual con Ubuntu 16.04.

De esta manera conseguimos:

  • Aislar nuestro código del resto del ordenador, sin que interfiera lo que pueda instalarse antes o después
  • Conseguir un entorno idéntico para todos los participantes del curso
  • Tener nuestra aplicación lista para subirla a la nube en un IAAS

Para esto lo más fácil es usar Vagrant, un software para facilitar la instalación, configuración y provisionamento de máquinas virtuales.

Lo haremos siguiendo la página Ubuntu 16.04 Xenial Guest on Vagrant – vagrant init bento/ubuntu-16.04
(la distribución oficial de ubuntu 16.04 no funciona del todo bien con vagrant) 

# Instalamos primero virtualbox y vagrant en el host
# (en ubuntu:)
sudo apt-get install virtualbox
sudo apt-get install vagrant

# Creamos el directorio para alojar la máquina virtual (MV):
mkdir DAI
cd DAI

# Creamos el archivo de configuración Vagrantfile ;
# desde este archivo podremos instalar interfaces, abrir puertos,
# crear carpetas compartidas, ajustar la memoria de la MV, instalar paquetes, etc
DAI> vagrant init bento/ubuntu-16.04


# Editamos primero el archivo Vagrantfile para definir las carpetas compartidas, y los puertos abiertos de la MV
#Ahora instalamos la MV
DAI> vagrant up

# y ya podemos conectarnos
DAI&gy; vagrant ssh

# parar la MV
DAI> vagrant halt

# o borrarla
DAI> vagrant destroy
(*) es recomendable una conexión a internet rápida para la instalación
